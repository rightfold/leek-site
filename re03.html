Name

ENUMERATE — return all entries in a journal

Synopsis

ENUMERATE journal

Description

ENUMERATE returns all entries in a journal, in order of appending.

Options

journal

The name of the journal to enumerate entries from. The journal must already exist.

Response

ENUMERATE returns one response part for each entry, and nothing else.
