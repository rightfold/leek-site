Name

FOLDL — reduce a journal to a single value

Synopsis

FOLDL journal
USING name
FROM extension

Description

FOLDL traverses a journal, keeping an accumulator that is updated by an extension, and finally returns the accumulator.

Options

journal

The name of the journal to traverse. The journal must already exist.

name

The name of the program from the extension to use. For some extensions (such as Lua), this is not the name but the source code of the program.

extension

The extension to call the program from.

Response

FOLDL returns a single response part with the final accumulator, and nothing else.
