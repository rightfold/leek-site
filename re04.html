Name

LOAD — load an extension

Synopsis

LOAD extension
FROM path

Description

LOAD loads an extension.

Options

extension

The name of the extension to load. This name will be expected in commands such as FOLDL. It is an error to attempt to load an extension with the same name as an already-loaded extension.

path

The path to the shared object that implements the extension on the server filesystem.

Response

LOAD returns an empty response upon success.
